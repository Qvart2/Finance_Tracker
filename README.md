# 💰 Финансовый трекер

Веб-приложение для учёта личных финансов на Flask.

## 🚀 Возможности

- **📝 Управление транзакциями** - добавление, редактирование и удаление доходов/расходов
- **📁 Категории** - создание собственных категорий с цветами для классификации транзакций
- **📊 Бюджеты** - установка месячных лимитов расходов по категориям
- **📈 Статистика** - отображение общего дохода, расходов и баланса
- **🔐 Авторизация** - защищённый доступ для каждого пользователя

## 🛠 Технологии

- **Backend**: Python, Flask
- **База данных**: SQLite, SQLAlchemy
- **Авторизация**: Flask-Login
- **Формы**: WTForms
- **Frontend**: HTML, CSS, JavaScript (Jinja2)

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Qvart2/Finance_Tracker.git
cd Finance_Tracker
```

2. Создайте виртуальное окружение:
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Запустите приложение:
```bash
python app.py
```

5. Откройте в браузере: http://localhost:5000

## 📖 Использование

### Регистрация и вход
1. На главной странице нажмите "Регистрация"
2. Введите имя пользователя и пароль
3. После регистрации автоматически создаются базовые категории

### Транзакции
- На странице "Транзакции" отображаются все записи
- Кнопка "+ Добавить" создаёт новую транзакцию
- Каждая транзакция содержит: сумму, дату, категорию и описание
- Редактирование и удаление доступны через соответствующие кнопки

### Категории
- Раздел "Категории" позволяет создавать собственные категории
- Каждая категория имеет название, тип (доход/расход) и цвет
- Категории можно редактировать и удалять
- При удалении категории транзакции сохраняются

### Бюджеты
- Бюджеты устанавливают месячные лимиты на категории расходов
- На странице бюджетов отображается прогресс использования
- Индикатор показывает оставшуюся сумму или превышение лимита
