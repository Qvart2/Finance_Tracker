# 💰 Финансовый трекер

Веб-приложение для учёта личных финансов на Flask.

## 📋 Возможности

- **Управление транзакциями**: добавление, редактирование и удаление доходов/расходов
- **Категории**: создание категорий с выбором цвета для классификации транзакций
- **Бюджеты**: установка месячных лимитов на категории расходов с отслеживанием остатка
- **Статистика**: визуальное отображение доходов, расходов и баланса
- **Авторизация**: регистрация и аутентификация пользователей

## 🛠 Технологический стек

- Python 3
- Flask
- Flask-SQLAlchemy
- Flask-Login
- SQLite
- HTML + CSS + JavaScript
- Jinja2 Templates

## 🚀 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Qvart2/Finance_Tracker.git
cd Finance_Tracker
```

### 2. Создание виртуального окружения

```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

```bash
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000

## 📖 Использование

### Регистрация и вход
1. Перейдите на страницу регистрации
2. Создайте аккаунт
3. Войдите в систему

### Управление транзакциями
1. Нажмите "+ Добавить транзакцию" на главной странице
2. Заполните форму: тип (доход/расход), сумма, дата, категория, описание
3. Сохраните транзакцию

### Управление категориями
1. Перейдите в раздел "Категории"
2. Создайте категории для группировки транзакций
3. Выберите цвет для визуального отображения

### Управление бюджетами
1. Перейдите в раздел "Бюджеты"
2. Установите месячный лимит для категории расходов
3. Отслеживайте расходы и остаток бюджета

## 🔧 Конфигурация

Секретный ключ для Flask можно изменить в `app.py`:

```python
app.config["SECRET_KEY"] = "ваш-секретный-ключ"
```
